Full Packet Capture (FPC) - Captures the entire packet, including the header and the payload, for all traffic.

Flow Collector - Means of recording network traffic's metadata and statistics.

NetFlow - Cisco-developed means of reporting network flow information to a structured database. Some key data that is captured includes:
- Network protocol interface
- Version and type of IP
- Source and destination IP address
- Source and destination port
- IP's type of service

Zeek (Bro) - Hybrid tool that passively monitors a network like a sniffer and only logs data of potential interest. Performs data normalization and then stores it as tab-delimited or JSON-formatted text files.

Multi Router Traffic Grapher (MRTG) - Creates graphs showing traffic flows through the network interfaces of routers and switches by polling the appliances using SNMP.
